# 🚀 快速开始 - 5 步完成环境搭建

## 步骤 1：安装依赖（2 分钟）

```bash
cd blog_system
pip install -r requirements.txt
```

## 步骤 2：检查数据库配置（1 分钟）

打开 `config.py`，确认数据库连接信息：

```python
# 修改这里的数据库信息
SQLALCHEMY_DATABASE_URI = 'mysql+pymysql://root:root123@localhost:3306/blog_system'
```

## 步骤 3：创建数据库（1 分钟）

在 MySQL 中执行：

```sql
CREATE DATABASE IF NOT EXISTS blog_system;
```

## 步骤 4：初始化数据库表（1 分钟）

```bash
python init_db.py
```

看到 "✅ 数据库表创建成功！" 就对了！

## 步骤 5：启动服务（1 分钟）

```bash
python app.py
```

看到 "🚀 服务运行在: http://127.0.0.1:5000" 就成功了！

---

## ✅ 验证是否成功
